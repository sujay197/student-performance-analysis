# Student Performance Analysis

This project analyzes student marks using Python.

## Features
- Calculates average marks
- Identifies top performer
- Displays subject-wise comparison graph
- Determines **Pass/Fail result** based on average marks

## Technologies Used
- Python
- Pandas
- Matplotlib

## How to Run
1. Install dependencies:
   python -m pip install pandas matplotlib

2. Run the script:
   python analysis.py
